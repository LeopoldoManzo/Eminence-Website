Eminence a simple and clean one page html5 template for independent (purchased) use ONLY built with valid HTML5 & CSS3. This site is based on latest Bootstrap framework 3.3.1 fully responsive web compatible with multi browser and devices. This template cannot be used for any needs like business, consultancy, agency, personal portfolio, profile, mobile website and startup company without written consent.


Key features
-------------
Twitter Bootstrap 3.3.1
Clean & Developer-friendly HTML5 and CSS3 code
100% Responsive Layout Design
One Page
Multipurpose theme
Google Fonts Support
Font Awesome 
Smooth Scrolling 
Fully Customizable
Contact Form
